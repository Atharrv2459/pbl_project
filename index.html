Employee Management & Geotracking System

"A full-stack web-based system to automate employee attendance, leave management, manual corrections, employee manager workflow, eliminate fraudulent attendance and location-based verification with role-based dashboards for employees and managers."

Problem Statement

Traditional management systems are prone to manual errors, fraudulent attendance, lack of transparency, and poor tracking of work hours, breaks, and leaves. Organizations need a secure, automated, and location-verified system to ensure accurate attendance, improve productivity, location tracking during work hours and simplify HR workflows.

Literature Review

Manual registers, basic biometric systems, and spreadsheet-based tracking lack real-time monitoring, analytics and location verification of employees.

Innovation

A unified platform with role-based dashboards, automated calculations, leave workflows, manual correction approvals, and geolocation-based attendance validation. The system integrates geolocation tracking to ensure employees mark attendance only from authorized locations, increasing authenticity and preventing misuse. It also has full time geolocation tracking so that location of employees can be tracked easily during working hours.

System Methodology

Input / Data

User profiles, attendance logs, break records, leave requests, manual entries, and geolocation data stored in PostgreSQL.

System Architecture

The system follows a modular full-stack architecture using React + Tailwind for frontend, Node.js + Express for backend, PostgreSQL for data storage, JWT for authentication, WebAuthn for device-bound biometric verification, REST APIs for communication, and OpenStreetMap for geolocation validation.

1. User logs in using JWT-based authentication.
2. User registers device once using system biometrics (WebAuthn).
3. During Punch-In, system verifies registered device + biometric.
4. Location is validated using geolocation & map services.
5. Attendance, breaks, leaves, and manual entries are stored in PostgreSQL.
6. Managers review dashboards, approvals, and analytics in real time.

Results & Analysis

Efficiency Improvement +30%

The system reduces manual errors, improves attendance accuracy, automates leave approval, eliminates fraudulent attendance, speeds up approval workflows, and provides real-time location visibility to managers compared to traditional manual or semi-automated systems.
